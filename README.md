**Sentence Splitter**

A simple algorithm that splits a sentence into a list of words.

Features
	•	Takes a sentence as input
	•	Trims unnecessary spaces
	•	Splits the sentence into individual words
	•	Returns a clean, ready-to-use list/array

**Projet structure**
.
├── src/
│   └── split.py
├── tests/
│   └── test_split.py
└── README.md

**Test**

The repository includes unit tests ensuring that:
	•	Multiple spaces are handled correctly
	•	An empty string returns an empty list
	•	Words are properly separated
